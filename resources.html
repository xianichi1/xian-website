Coding Resources Used

  • MDN Web Docs – HTML: Used for structuring all pages and implementing form elements, headings, lists, and semantic tags.
  • MDN Web Docs – CSS: Referenced for styling layouts, responsive design, dropdown navigation, sidebar components, and custom animations.
  • MDN Web Docs – JavaScript: Used for dropdown toggling, interactive “Catch the Mouse” game logic, and event listeners.
  • Google Fonts: Used for the ‘Lexend’ and ‘Darumadrop One’ typefaces implemented via embedded links in the head section of each HTML page.
  • W3Schools: Used for quick lookups on form handling, dropdown menus, and file upload input for the image submission form.
  • CSS-Tricks: Referenced for animation timing, flexbox layout structure, and styling input fields and dropdowns.
  • Stack Overflow: Consulted for troubleshooting JavaScript behaviors, such as resetting dropdowns and handling image loading in interactive components.
  • Wikimedia Commons: Provided the placeholder cartoon mouse image for the interactive game element.

Cat Information Sources

  • DeziRoo: Used for info on why cats chase their humans.
  • AAHA: Reference for kneading behavior and its meanings.
  • Three Chatty Cats: Source for enrichment ideas like cat shelves.
  • Hervey Cats Foundation: Helped shape the explanation of cat recognition and attachment.
  • Tuft + Paw: Provided guidance for interpreting cat body language and signals.
  • PetRadar: Contributed to the behavioral section and tips on training cats.
